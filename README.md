# MILK FILTER ONLINE

![](<images/Screenshot 2026-06-26 at 13-02-03 MILK FILTER ONLINE.png>)

A web-based image and video filter tool heavily inspired by [Milk inside a bag of milk inside a bag of milk](https://nikita-kryukov.itch.io/pmkm) by Nikita Kryukov — both in aesthetic and in spirit. The UI borrows directly from the game's visual language: the pixel borders, the window frames, the dark cramped atmosphere, even the cursor.

> *"I'd like to buy a bag of milk."*

---

## Filters

### Pixel Art Filter
A dithering-based pixel art filter with custom color palettes made by me, all inspired by the game's color palette. Adjustable brightness, contrast, pixel scale and grain strength.

### Milk Filter
A port of the [Milk Filter](https://github.com/LucaSinUnaS/Milk-Filter) originally created by **[LucaSinUnaS](https://github.com/LucaSinUnaS)**, adapted here as a client-side web filter. Full credit goes to him for the original algorithm and concept. Supports multiple effect presets, pointillism and compression modes.

## Features

- Two independent filters: Pixel Art (dither) and Milk
- Drag & drop, paste, or file picker — images and video
- Adjustable parameters per filter
- Copy result or save as PNG
- EN / ES interface
- Pixel-perfect UI assets from the game's visual style

---

## Inspiration & Context

This project exists somewhere between fan tool and tribute. The entire visual identity — the dashed pixel borders around panels, the window chrome, the deep red-on-black color scheme, the Press Start 2P font — is a deliberate echo of the world Nikita built.

If you haven't played the game, start there: [nikita-kryukov.itch.io/pmkm](https://nikita-kryukov.itch.io/pmkm)

There is also a community ARG that lives at **[passwordpassword.online](https://passwordpassword.online/)** — a rabbit hole tied to the game's universe. This tool fits somewhere in that world too.
